# Быстрый старт AntiZapret

## Одна команда для установки

```bash
cd /root && git clone https://github.com/Limych/antizapret.git && cd antizapret && chmod +x install.sh && ./install.sh
```

## Что делает скрипт

1. Определяет вашу систему FreeBSD
2. Находит локальный IP адрес роутера
3. Скачивает актуальные версии пакетов:
   - Tor
   - obfs4proxy (для обхода блокировок Tor)
   - webtunnel (альтернативный транспорт)
4. Запрашивает у вас bridge-линии (мосты)
5. Настраивает Tor с правильной конфигурацией
6. Устанавливает автозапуск
7. Настраивает обновление списков заблокированных IP
8. Запускает Tor

## Подготовка

Перед запуском получите мосты:

### Вариант 1: Через браузер
Зайдите на https://bridges.torproject.org/ и получите:
- obfs4 мосты
- webtunnel мосты

### Вариант 2: Через email
Отправьте письмо на bridges@torproject.org с темой:
```
get transport obfs4
```

## После установки

1. Откройте веб-интерфейс OPNsense
2. Создайте алиас для IP списка
3. Настройте NAT правило
4. Добавьте задачу cron для обновления

Подробные инструкции скрипт выведет в конце установки.

## Проверка

```bash
ps aux | grep tor
tail -f /var/log/tor/notices.log
```
