# История изменений

## [Новые возможности] - 2025-12-15

### Добавлено
- Интерактивный скрипт автоматической установки `install.sh`
  - Автоматическое определение системы и архитектуры
  - Автоматический поиск актуальных версий пакетов из репозитория FreeBSD
  - Автоматическое определение локального IP адреса
  - Интерактивный ввод bridge-линий
  - Красивый цветной вывод с индикаторами прогресса
  - Автоматическая настройка всех компонентов
  - Интеграция с OPNsense

- Поддержка obfs4proxy и webtunnel для обхода блокировок Tor
  - Автоматическая установка obfs4proxy-tor
  - Автоматическая установка webtunnel-tor
  - Настройка ClientTransportPlugin в torrc

- Улучшенная документация
  - QUICKSTART.md - быстрый старт для нетерпеливых
  - INSTALL.md - подробная инструкция по установке
  - FAQ.md - часто задаваемые вопросы и решения
  - DEMO.txt - пример работы скрипта

### Изменено
- Обновлен README.md с информацией об автоматической установке
- Обновлен TOR.md с инструкциями по использованию obfs4/webtunnel
- Улучшена структура документации

### Технические детали
- Скрипт совместим с FreeBSD 13+ и OPNsense 21+
- Автоматический поиск пакетов в репозитории FreeBSD
- Fallback на известные версии пакетов при проблемах с сетью
- Проверка синтаксиса shell скрипта
- Spinner-индикаторы для длительных операций

## [Предыдущие версии]

### Существующая функциональность
- Perl скрипт для загрузки и обработки списков заблокированных IP
- Поддержка группировки IP адресов в подсети /24
- Интеграция с OPNsense через actions.d
- Автоматическое обновление списков через cron
- Поддержка белого списка (white.list)
- Генератор заметок к релизам (bin/gen_releasenotes)

### Документированные методы установки
- Ручная установка с Tor (TOR.md)
- Ручная установка с WireGuard VPN (WireGuard.md)
- Настройка правил файрвола
- Настройка NAT и алиасов
