# Возможности AntiZapret

## Основные функции

### Прозрачный обход блокировок
- Работает на уровне роутера
- Не требует настройки клиентских устройств
- Поддержка HTTP, HTTPS и любого другого TCP трафика
- Автоматическое определение заблокированных ресурсов

### Поддерживаемые технологии

#### Tor с pluggable transports
- obfs4proxy - обфускация Tor трафика
- webtunnel - имитация обычного HTTPS трафика
- Автоматическая настройка мостов (bridges)
- Исключение узлов из определенных стран

#### WireGuard VPN
- Альтернатива Tor для тех, у кого есть внешний VPN сервер
- Высокая скорость работы
- Простая настройка

## Автоматизация

### Интерактивный скрипт установки
- Определение системы и архитектуры
- Поиск актуальных версий пакетов
- Автоматическое определение локального IP
- Интерактивный ввод настроек
- Цветной вывод с индикаторами прогресса
- Полная настройка за несколько минут

### Автоматическое обновление
- Ежедневное обновление списков блокировок
- Интеграция с cron
- Скачивание данных из официального источника zapret-info
- Группировка IP адресов в подсети для оптимизации

### Обработка списков

#### Умная оптимизация IP адресов
- Группировка в подсети /24 при наличии 10+ адресов
- Удаление дубликатов
- Сортировка для быстрого поиска

#### Белый список (whitelist)
- Возможность добавить дополнительные домены и IP
- Поддержка IPv4 адресов и доменов
- Автоматическое объединение со списком блокировок

## Интеграция

### OPNsense
- Автоматическая настройка через configd
- Интеграция с системой cron
- Готовые actions для обновления списков
- Поддержка алиасов и NAT правил

### pfSense
- Совместимость с базовыми функциями
- Использование стандартных возможностей pf

### Файрвол
- Использование pf (packet filter)
- Поддержка таблиц (tables) для быстрого поиска
- NAT перенаправление на прозрачный прокси
- Минимальная нагрузка на систему

## Безопасность

### Приватность
- Весь трафик через Tor зашифрован
- Обфускация для обхода DPI (Deep Packet Inspection)
- Исключение узлов из небезопасных стран
- Отсутствие логирования пользовательских запросов

### Надежность
- Поддержка нескольких мостов одновременно
- Автоматическое переключение при сбоях
- Резервные URL для загрузки пакетов
- Проверка синтаксиса конфигурации
